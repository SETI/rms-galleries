# Plains - False Color * Click the [image above](/press_releases/medium/PIA22xxx/PIA22786_med.jpg) for a larger view * [Full-Res JPEG](https://photojournal.jpl.nasa.gov/jpeg/PIA22786.jpg) (624 x 2701) (149.9 kB) * [Full-Res TIFF](https://photojournal.jpl.nasa.gov/tiff/PIA22786.tif) (624 x 2701) (2.4 MB)

Caption:


Context image

This VIS image shows the plains to the east of Hecates Tholus.

The THEMIS VIS camera contains 5 filters. The data from different filters can be combined in multiple ways to create a false color image. These false color images may reveal subtle variations of the surface not easily identified in a single band image.
