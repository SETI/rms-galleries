# Dust Devil Tracks * Click the [image above](/press_releases/medium/PIA22xxx/PIA22987_med.jpg) for a larger view * [Full-Res JPEG](https://photojournal.jpl.nasa.gov/jpeg/PIA22987.jpg) (1340 x 2690) (514.2 kB) * [Full-Res TIFF](https://photojournal.jpl.nasa.gov/tiff/PIA22987.tif) (1340 x 2690) (2.9 MB)

Caption:


Context image

This VIS image shows a region dense with dust devil tracks. Located in Promethei Terra, these tracks run parallel to the nearby southeastern margin of Hellas Planitia. The number of tracks and amount of overlapping indicates that there are long term winds in this region.
