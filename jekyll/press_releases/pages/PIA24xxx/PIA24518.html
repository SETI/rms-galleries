# Holden Crater Rim - False Color * Click the [image above](/press_releases/medium/PIA24xxx/PIA24518_med.jpg) for a larger view * [Full-Res JPEG](https://photojournal.jpl.nasa.gov/jpeg/PIA24518.jpg) (672 x 1425) (97.9 kB) * [Full-Res TIFF](https://photojournal.jpl.nasa.gov/tiff/PIA24518.tif) (672 x 1425) (2.0 MB)

Caption:


Context image

The THEMIS VIS camera contains 5 filters. The data from different filters can be combined in multiple ways to create a false color image. These false color images may reveal subtle variations of the surface not easily identified in a single band image. Today's false color image shows part of the rim of Holden Crater. Located in Noachis Terra, Holden Crater is 152km (94 miles) in diameter.
