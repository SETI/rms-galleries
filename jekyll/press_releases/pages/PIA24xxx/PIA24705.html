# Soffen Crater - False Color * Click the [image above](/press_releases/medium/PIA24xxx/PIA24705_med.jpg) for a larger view * [Full-Res JPEG](https://photojournal.jpl.nasa.gov/jpeg/PIA24705.jpg) (594 x 2720) (118.8 kB) * [Full-Res TIFF](https://photojournal.jpl.nasa.gov/tiff/PIA24705.tif) (594 x 2720) (2.2 MB)

Caption:


Context image

The THEMIS VIS camera contains 5 filters. The data from different filters can be combined in multiple ways to create a false color image. These false color images may reveal subtle variations of the surface not easily identified in a single band image. Today's false color image shows a cross section of Soffen Crater. Located in Terra Cimmeria, Soffen Crater is 58km (36 miles) in diameter.
