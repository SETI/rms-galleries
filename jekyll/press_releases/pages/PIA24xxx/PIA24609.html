# Morning Clouds - False Color * Click the [image above](/press_releases/medium/PIA24xxx/PIA24609_med.jpg) for a larger view * [Full-Res JPEG](https://photojournal.jpl.nasa.gov/jpeg/PIA24609.jpg) (694 x 1428) (64.3 kB) * [Full-Res TIFF](https://photojournal.jpl.nasa.gov/tiff/PIA24609.tif) (694 x 1428) (1.9 MB)

Caption:


Context image

The THEMIS VIS camera contains 5 filters. The data from different filters can be combined in multiple ways to create a false color image. These false color images may reveal subtle variations of the surface not easily identified in a single band image. Today's false color image shows part of the extensive Tharsis volcanic fields. The pale wispy spots are clouds. This image was collected just after dawn during the spring season.
