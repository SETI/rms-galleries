# Claritas Fossae Channel * Click the [image above](/press_releases/medium/PIA23xxx/PIA23075_med.jpg) for a larger view * [Full-Res JPEG](https://photojournal.jpl.nasa.gov/jpeg/PIA23075.jpg) (1314 x 2662) (277.7 kB) * [Full-Res TIFF](https://photojournal.jpl.nasa.gov/tiff/PIA23075.tif) (1314 x 2662) (2.5 MB)

Caption:


Context image

Today's VIS image shows a section of an unnamed channel. This channel starts in Claritas Fossae and empties down into Icaria Planum. The linear features that parallel the sides of the image are fractures that can create linear depressions (fossae).
