# Noachis Terra - False Color * Click the [image above](/press_releases/medium/PIA23xxx/PIA23252_med.jpg) for a larger view * [Full-Res JPEG](https://photojournal.jpl.nasa.gov/jpeg/PIA23252.jpg) (688 x 1428) (93.1 kB) * [Full-Res TIFF](https://photojournal.jpl.nasa.gov/tiff/PIA23252.tif) (688 x 1428) (1.9 MB)

Caption:


Context image

Today's image shows the floor of an unnamed crater in Noachis Terra.

The THEMIS VIS camera contains 5 filters. The data from different filters can be combined in multiple ways to create a false color image. These false color images may reveal subtle variations of the surface not easily identified in a single band image.
