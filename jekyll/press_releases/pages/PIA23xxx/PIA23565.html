# Kasei Valles * Click the [image above](/press_releases/medium/PIA23xxx/PIA23565_med.jpg) for a larger view * [Full-Res JPEG](https://photojournal.jpl.nasa.gov/jpeg/PIA23565.jpg) (1391 x 2642) (209.1 kB) * [Full-Res TIFF](https://photojournal.jpl.nasa.gov/tiff/PIA23565.tif) (1391 x 2642) (2.5 MB)

Caption:


Context image

This VIS image shows a portion of Kasei Valles. In the region of this image Kasei Valles is flowing northward from Vallis Marineris. This image contains both fluvial features and tectonic features. The linear features running across the image from the lower right to the upper left are fractures created by tectonic forces. The ones running vertically were created by fluid flow. Kasei Valles is one of the largest outflow channels on Mars and catastrophic flooding carved this massive channel.
